# Brainstorm Workflow (Fast Capture)

Use this process so ideas are captured fast and organized later.

## 1) Quick Capture (no pressure, no formatting required)

Drop thoughts in plain language. Even fragments are fine.

Examples:
- "What if we detect tailings from LiDAR?"
- "Could we score missed pockets around old claims?"
- "Need a better way to rank field targets."

## 2) Organizer Pass (assistant does this)

For each captured idea, place it in exactly one status:
- `ToDo` (decided YES)
- `Nixed` (decided NO)
- `Undecided/Deferred` (not discussed, not decided, or intentionally put off)

Every idea entry must include:
- Idea
- Goal
- Method
- Output
- Status
- Reason (why yes/no/deferred)
- Date logged
- Source context
- Next discussion trigger

## 3) Session Checkpoint

At any time, ask:
- "What do we have so far?"
- "Show ToDo / Nixed / Deferred with reasons."
- "What is still unclear?"

## 4) Decision Rule

If no explicit decision is made, default to `Undecided/Deferred` (never assume yes/no).
